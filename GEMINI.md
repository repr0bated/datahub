# DataHub Gemini Context

## Focus Area
**Metadata Catalog**

## Why Index?
Standard for managing large-scale metadata, useful for the `op-introspection` indexer.

## Project Description
The Data Discovery Platform for the Modern Data Stack. It provides end-to-end data lineage, search, and discovery.

## Key Concepts
- **Metadata Service (GMS)**: The core backend.
- **Ingestion**: Python-based framework to extract metadata.
- **Frontend**: React-based UI.
- **Models**: PDL (Pegasus Data Language) defined models.

## Conventions
- **Language**: Java, Python, TypeScript, GraphQL.
- **Style**: Microservices architecture.
